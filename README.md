# 🎬 FilmLibrary

Простое десктопное приложение для управления фильмами и сериалами. Позволяет добавлять, просматривать и удалять фильмы с отображением изображений и описаний. 

Технологии:
C#, WPF, .NET, SQLite, Entity Framework Core, MaterialDesignThemes 

Функции:   
- Добавление и удаление фильмов  
- Просмотр деталей с изображением  
- Локальное хранение данных
     

## Скриншоты приложения
